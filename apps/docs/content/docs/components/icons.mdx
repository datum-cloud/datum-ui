## Overview

The icon system provides three components: `Icon` for rendering any Lucide icon with consistent defaults, `CloseIcon` for a styled dismissal indicator, and `SpinnerIcon` for animated loading states. The `Icon` wrapper standardizes stroke width and sizing across the application.

## Usage

The `Icon` component accepts any Lucide icon via the `icon` prop:

```tsx
import { Icon } from '@datum-cloud/datum-ui/icon'
import { Search } from 'lucide-react'

<Icon icon={Search} />
```

## Examples

### Lucide Icons with Icon Wrapper

The `Icon` component accepts any Lucide icon and applies consistent defaults for `strokeWidth`, `absoluteStrokeWidth`, and `size`.

```tsx
import { Icon } from '@datum-cloud/datum-ui/icon'
import { Search, Settings, Bell, User, Plus, Trash2, Heart, ChevronRight } from 'lucide-react'

<Icon icon={Search} />
<Icon icon={Settings} />
<Icon icon={Bell} />
<Icon icon={User} />
```

### Custom Sizes

Override the default `size` prop to render icons at different dimensions.

```tsx
<Icon icon={Settings} size={12} />
<Icon icon={Settings} size={16} />
<Icon icon={Settings} size={20} />
<Icon icon={Settings} size={24} />
<Icon icon={Settings} size={32} />
```

### Custom Stroke Width

Adjust the `strokeWidth` for thinner or bolder icon lines.

```tsx
<Icon icon={Heart} size={24} strokeWidth={0.5} absoluteStrokeWidth={false} />
<Icon icon={Heart} size={24} strokeWidth={1} absoluteStrokeWidth={false} />
<Icon icon={Heart} size={24} strokeWidth={1.5} absoluteStrokeWidth={false} />
<Icon icon={Heart} size={24} strokeWidth={2} absoluteStrokeWidth={false} />
```

## Props

### Icon

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `LucideIcon` | -- | The Lucide icon component to render |
| `size` | `number` | `16` | Icon dimensions in pixels |
| `strokeWidth` | `number` | `1` | Stroke width of the icon lines |
| `absoluteStrokeWidth` | `boolean` | `true` | Whether stroke width is absolute (not scaled with size) |
| `...props` | `LucideProps` | -- | All standard Lucide icon props |

### CloseIcon

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes for the SVG element |
| `fill` | `string` | `'fill-foreground'` | Tailwind fill class for the icon color |

### SpinnerIcon

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl' \| '5xl' \| '6xl' \| '7xl' \| '8xl' \| '9xl'` | `'md'` | Size of the spinner |
| `className` | `string` | -- | Additional CSS classes |
| `trackClassName` | `string` | `'text-border'` | Color class for the background track circle |
| `indicatorClassName` | `string` | `'text-primary'` | Color class for the spinning indicator arc |
| `...props` | `React.SVGProps<SVGSVGElement>` | -- | All standard SVG element props |
