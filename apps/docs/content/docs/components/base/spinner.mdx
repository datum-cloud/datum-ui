## Overview

SpinnerIcon renders an animated SVG spinner with a track circle and a spinning arc indicator. It is used inside buttons during loading states and as standalone loading indicators. The component supports ten size presets and customizable track/indicator colors.

## Dependencies

No additional dependencies required — this component uses only React and Tailwind CSS.

## Usage

<ComponentPreview name="Basic Spinner">
  <SpinnerIcon />
</ComponentPreview>

```tsx
import { SpinnerIcon } from '@datum-cloud/datum-ui'

<SpinnerIcon />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl' \| '5xl' \| '6xl' \| '7xl' \| '8xl' \| '9xl'` | `'md'` | Spinner size |
| `trackClassName` | `string` | `'text-border'` | Tailwind color class for the background circle |
| `indicatorClassName` | `string` | `'text-primary'` | Tailwind color class for the spinning arc |
| `className` | `string` | -- | Additional CSS classes |
