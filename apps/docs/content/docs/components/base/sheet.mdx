## Overview

Sheet is a panel that slides in from a screen edge (right, left, top, or bottom). It is built on Radix UI's Dialog primitive and is useful for settings panels, detail views, and mobile navigation menus without navigating away from the current page.

## Examples

### Side Variants

The `side` prop controls which edge the sheet slides in from.

```tsx
<SheetContent side="left">...</SheetContent>
<SheetContent side="right">...</SheetContent>
<SheetContent side="top">...</SheetContent>
<SheetContent side="bottom">...</SheetContent>
```

## Props

### SheetContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `'top' \| 'right' \| 'bottom' \| 'left'` | `'right'` | Edge the sheet slides in from |
| `className` | `string` | -- | Additional CSS classes |

### Sub-components

`Sheet`, `SheetTrigger`, `SheetClose`, `SheetHeader`, `SheetFooter`, `SheetTitle`, and `SheetDescription` follow the same trigger + content composition pattern as Dialog.
