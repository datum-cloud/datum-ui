import { RadioGroup, RadioGroupItem, Label } from '@/components/ui'

## Overview

The `RadioGroup` component renders a group of radio buttons where only one option can be selected at a time. It uses Radix UI RadioGroup for accessible keyboard navigation and focus management.

## Dependencies

This component requires the following packages:

```package-install
@radix-ui/react-radio-group
```

## Usage

## Props

### RadioGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | -- | Controlled selected value |
| `defaultValue` | `string` | -- | Default selected value |
| `onValueChange` | `(value: string) => void` | -- | Callback when selection changes |
| `disabled` | `boolean` | `false` | Disables all items |
| `required` | `boolean` | `false` | Marks the group as required |
| `orientation` | `'horizontal' \| 'vertical'` | `'vertical'` | Layout direction |
| `name` | `string` | -- | Name for form submission |
| `className` | `string` | -- | Additional CSS classes |

### RadioGroupItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | -- | Unique value for this option |
| `disabled` | `boolean` | `false` | Disables this item |
| `id` | `string` | -- | ID for label association |
| `className` | `string` | -- | Additional CSS classes |
