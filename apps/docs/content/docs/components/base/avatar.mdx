import { Avatar, AvatarImage, AvatarFallback } from '@/components/ui'

## Overview

Avatar displays a user's profile picture in a circular frame with built-in fallback handling. When an image fails to load or is not provided, a fallback element (typically initials or an icon) is displayed instead.

## Dependencies

This component uses `@radix-ui/react-avatar` for accessibility and fallback handling.

## Usage

<ComponentPreview name="Basic Avatar">
  <Avatar>
    <AvatarImage src="https://i.pravatar.cc/150?u=avatar1" alt="User Avatar" />
    <AvatarFallback>AB</AvatarFallback>
  </Avatar>
</ComponentPreview>


## Components

### Avatar

The root container for the avatar. By default, it renders as a circular frame with size-8 (32px).

### AvatarImage

The image element to display within the avatar. It supports all standard `img` attributes. When the image fails to load, it automatically falls back to the `AvatarFallback`.

### AvatarFallback

The fallback content displayed when no image is provided or when the image fails to load. Typically contains user initials or an icon.

## Props

### Avatar Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Additional CSS classes for the avatar container |
| `...props` | `React.ComponentProps<typeof AvatarPrimitive.Root>` | -- | All Radix UI Avatar.Root props |

### AvatarImage Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | -- | Image URL to display |
| `alt` | `string` | -- | Alternative text for the image |
| `className` | `string` | -- | Additional CSS classes for the image |
| `...props` | `React.ComponentProps<typeof AvatarPrimitive.Image>` | -- | All Radix UI Avatar.Image props |

### AvatarFallback Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | -- | Content to display as fallback (e.g., initials, icon) |
| `className` | `string` | -- | Additional CSS classes for the fallback |
| `delayMs` | `number` | -- | Delay in milliseconds before showing the fallback |
| `...props` | `React.ComponentProps<typeof AvatarPrimitive.Fallback>` | -- | All Radix UI Avatar.Fallback props |

## Accessibility

- The component uses Radix UI's Avatar primitive, which provides built-in accessibility features
- Always provide an `alt` attribute for `AvatarImage` to ensure screen readers can describe the image
- Fallback content should be meaningful (e.g., user initials) for screen reader users
