import { Skeleton } from '@/components/ui'

## Overview

Skeleton renders a pulsing placeholder element that mimics the shape of content while it loads. Use it inside cards, lists, or any layout to provide visual feedback during data fetching.

## Dependencies

No additional dependencies required — this component uses only React and Tailwind CSS.

## Usage

<ComponentPreview name="Basic Skeleton">
  <Skeleton className="h-10 w-64" />
</ComponentPreview>

```tsx
import { Skeleton } from '@datum-cloud/datum-ui/skeleton'

<Skeleton className="h-10 w-64" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | -- | Set dimensions and shape (e.g. `h-4 w-full`, `rounded-full`) |
| `...props` | `React.HTMLAttributes<HTMLDivElement>` | -- | All standard div attributes |
