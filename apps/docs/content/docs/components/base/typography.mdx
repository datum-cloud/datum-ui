## Overview

The Typography module provides a set of semantic text components: `Title`, `Text`, `Paragraph`, `Link`, `List`, `Blockquote`, and `Code`. All components use CVA variants for consistent sizing, weight, and color. `Title` renders responsive heading elements, `Text` supports inline formatting shortcuts (bold, italic, code, mark, etc.), and `Paragraph` handles block-level text with spacing control.

## Usage

```tsx
import { Title, Text, Paragraph } from '@datum-cloud/datum-ui/typography'

<Title level={1}>Page Heading</Title>
<Paragraph>Body text goes here.</Paragraph>
<Text strong>Bold inline text</Text>
```

## Sub-components

| Component | Description |
|-----------|-------------|
| `Title` | Responsive heading element (h1–h6) with level, weight, and color variants. |
| `Text` | Inline text with size, weight, color, and formatting variants (bold, italic, code, etc.). |
| `Paragraph` | Block-level text with size and line-spacing variants. |
| `Link` | Styled anchor with focus ring. Auto-sets `rel="noopener noreferrer"` for `target="_blank"`. |
| `List` | Ordered or unordered list container. |
| `ListItem` | List item. |
| `Blockquote` | Styled blockquote with left border accent. |
| `Code` | Inline code or pre-formatted code block. |

## Props — Title

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `level` | `1 \| 2 \| 3 \| 4 \| 5 \| 6` | `4` | Heading size (responsive). Also determines the default HTML element. |
| `as` | `'h1' \| 'h2' \| ... \| 'h6'` | `h{level}` | Override the rendered HTML element. |
| `weight` | `'normal' \| 'medium' \| 'semibold' \| 'bold' \| 'extrabold'` | `'semibold'` | Font weight. |
| `textColor` | `'default' \| 'muted' \| 'primary' \| 'secondary' \| 'destructive' \| 'success' \| 'warning' \| 'info'` | `'default'` | Text color. |

## Props — Text

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xs' \| 'sm' \| 'base' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl'` | `'base'` | Font size. |
| `weight` | `'normal' \| 'medium' \| 'semibold' \| 'bold' \| 'extrabold'` | `'normal'` | Font weight. |
| `textColor` | Same as Title | `'default'` | Text color. |
| `as` | `'span' \| 'p' \| 'div'` | `'span'` | Rendered HTML element. |
| `copyable` | `boolean` | — | Show a copy button that copies children to clipboard. |
| `ellipsis` | `boolean` | — | Truncate with ellipsis on overflow. |
| `strong` | `boolean` | — | Shortcut for `type="strong"`. |
| `italic` | `boolean` | — | Shortcut for `type="italic"`. |
| `code` | `boolean` | — | Shortcut for `type="code"`. |
| `mark` | `boolean` | — | Shortcut for `type="mark"`. |
| `underline` | `boolean` | — | Shortcut for `type="underline"`. |
| `delete` | `boolean` | — | Shortcut for `type="delete"` (strikethrough). |

## Props — Paragraph

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xs' \| 'sm' \| 'base' \| 'lg' \| 'xl'` | `'base'` | Font size. |
| `spacing` | `'tight' \| 'normal' \| 'loose'` | `'normal'` | Line height. |
| `as` | `'p' \| 'div'` | `'p'` | Rendered HTML element. |

## Props — Link

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | — | **Required.** Destination URL. |
| `target` | `'_blank' \| '_self' \| '_parent' \| '_top'` | `'_self'` | Link target. |

## Props — List

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `listType` | `'ordered' \| 'unordered'` | `'unordered'` | List style (numbered or bulleted). |
| `as` | `'ol' \| 'ul'` | — | Override the rendered element (auto-determined by `listType`). |

## Props — Code

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `'code' \| 'pre'` | `'code'` | Inline code or pre-formatted block. |
