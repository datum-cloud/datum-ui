import { Textarea } from '@/components/ui'

## Overview

The `Textarea` component wraps a native `<textarea>` element with Datum styling. It uses `field-sizing: content` for automatic height adjustment and integrates with the Datum theming system for consistent focus and border styles.

## Dependencies

No additional dependencies required — this component uses only React and Tailwind CSS.

## Usage

<ComponentPreview name="Basic Textarea">
  <Textarea placeholder="Type your message..." />
</ComponentPreview>

```tsx
import { Textarea } from '@datum-cloud/datum-ui/textarea'

<Textarea placeholder="Type your message..." />
```

## Props

The `Textarea` component accepts all standard HTML `<textarea>` attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | -- | Placeholder text |
| `rows` | `number` | -- | Minimum number of visible rows |
| `disabled` | `boolean` | `false` | Disables the textarea |
| `value` | `string` | -- | Controlled value |
| `defaultValue` | `string` | -- | Uncontrolled default value |
| `onChange` | `(e: ChangeEvent) => void` | -- | Change callback |
| `className` | `string` | -- | Additional CSS classes |
| `ref` | `RefObject<HTMLTextAreaElement>` | -- | Ref forwarded to the `<textarea>` element |
