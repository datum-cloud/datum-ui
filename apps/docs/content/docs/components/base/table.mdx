import { Table, TableHeader, TableBody, TableFooter, TableHead, TableRow, TableCell, TableCaption } from '@/components/ui'

## Overview

Table provides unstyled, composable HTML table primitives with consistent Datum styling. Use it when you need full control over table markup.

## Dependencies

No additional dependencies required — this component uses only React and Tailwind CSS.

## Usage

<ComponentPreview name="Basic Table">
  <Table>
    <TableHeader>
      <TableRow>
        <TableHead>Name</TableHead>
        <TableHead>Email</TableHead>

      </TableRow>
    </TableHeader>
    <TableBody>
      <TableRow>
        <TableCell>Alice Johnson</TableCell>

        <TableCell>Bob Smith</TableCell>

  </Table>

## Sub-components

| Component | HTML Element | Description |
|-----------|-------------|-------------|
| `Table` | `<table>` | Root table wrapper with horizontal scroll container |
| `TableHeader` | `<thead>` | Table header group |
| `TableBody` | `<tbody>` | Table body group |
| `TableFooter` | `<tfoot>` | Table footer group |
| `TableHead` | `<th>` | Header cell |
| `TableRow` | `<tr>` | Table row with hover and selected states |
| `TableCell` | `<td>` | Body cell |
| `TableCaption` | `<caption>` | Table caption |

All sub-components accept their native HTML element props plus an optional `className` for custom styling.
