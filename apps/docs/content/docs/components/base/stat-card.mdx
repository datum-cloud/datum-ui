## Overview

Stat Card is a composition pattern using the Card, CardHeader, CardContent, CardTitle, and CardDescription components to display key metrics. There is no dedicated StatCard component -- instead, compose the existing Card primitives to build stat displays that match your layout needs.

## Dependencies

No additional dependencies required — this component uses only React and Tailwind CSS.

## Usage

<ComponentPreview name="Basic Stat Card">
  <Card className="w-64">
    <CardHeader>
      <CardDescription>Total Revenue</CardDescription>
      <CardTitle className="text-2xl font-bold">$45,231.89</CardTitle>
    </CardHeader>
    <CardContent>
      <p className="text-xs text-muted-foreground">+20.1% from last month</p>

  </Card>

```tsx
import {
  Card,
  CardHeader,
  CardContent,
  CardTitle,
  CardDescription,
} from '@datum-cloud/datum-ui/card'

<Card className="w-64">
  <CardHeader>
    <CardDescription>Total Revenue</CardDescription>

  </CardHeader>
  <CardContent>
    <p className="text-xs text-muted-foreground">+20.1% from last month</p>
  </CardContent>
</Card>
```

## Components Used

| Component | Description |
|-----------|-------------|
| `Card` | Root container with border and padding |
| `CardHeader` | Groups title and description |
| `CardTitle` | Primary display value (the stat number) |
| `CardDescription` | Label above the stat value |
| `CardContent` | Additional context below the header |
| `CardFooter` | Optional footer area |

All components accept a `className` prop for custom styling. See the [Card](/docs/components/card) documentation for full details.
