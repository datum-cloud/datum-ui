import { EmptyContent } from '@/components/ui'

## Overview

EmptyContent provides a branded empty state placeholder with decorative illustrations, a greeting message, optional subtitle, and action buttons. Use it inside tables, lists, or pages to guide users when there is no data to display.

## Dependencies

No additional dependencies required — this component uses only React and Tailwind CSS.

## Usage

<ComponentPreview name="Basic EmptyContent">
  <EmptyContent title="no items yet." />
</ComponentPreview>

```tsx
import { EmptyContent } from '@datum-cloud/datum-ui/empty-content'

<EmptyContent title="no items yet." />
```

## Examples

### With Subtitle and Action

<ComponentPreview name="EmptyContent With Action">

    actions={[

```tsx
<EmptyContent
  title="no projects found."
  subtitle="Create your first project to get started."
  actions={[
    {
      as: 'button',
      label: 'Create Project',

    },
  ]}
/>
```

### Link Actions

Actions can render as links or external links instead of buttons.

```tsx
<EmptyContent
  title="no documentation yet."
  actions={[
    {
      as: 'link',
      label: 'View Docs',
      to: '/docs',
    },
    {
      as: 'external-link',
      label: 'GitHub',
      to: 'https://github.com/datum-cloud',
    },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `'No data found'` | Main message (appended to greeting) |
| `subtitle` | `string` | -- | Secondary description text |
| `variant` | `'default' \| 'dashed' \| 'minimal'` | `'default'` | Border style |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Container and text size |
| `orientation` | `'vertical' \| 'horizontal'` | `'vertical'` | Layout direction |
| `spacing` | `'compact' \| 'normal' \| 'relaxed'` | `'normal'` | Action button spacing |
| `actions` | `EmptyContentAction[]` | `[]` | Action buttons or links |
| `userName` | `string` | `'there'` | Name used in greeting (e.g. "Hey Alice, ...") |
| `linkComponent` | `React.ElementType` | `'a'` | Custom link component (e.g. React Router's `Link`) |
| `className` | `string` | -- | Additional CSS class |

### EmptyContentAction

Actions are a discriminated union keyed on `as`. A `button` action accepts the full
[`Button`](/docs/components/base/button) prop surface; `link` / `external-link` actions
additionally accept anchor attributes plus `to`.

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `as` | `'button' \| 'link' \| 'external-link'` | `'button'` | Action kind (discriminator) |
| `label` | `string` | -- | Button or link text |
| `to` | `string` | -- | URL (required for `link` and `external-link`) |
| `disabled` | `boolean` | `false` | Disables the action; a disabled link renders as a plain disabled button |
| `hidden` | `boolean` | `false` | When true the action is not rendered (useful for RBAC) |
| `tooltip` | `ReactNode` | -- | Tooltip shown on hover/focus (works while disabled) |
| `tooltipSide` | `'top' \| 'right' \| 'bottom' \| 'left'` | -- | Tooltip placement |
| `type` | `'primary' \| 'secondary' \| 'danger' \| …` | `'secondary'` | Button color/semantic type (any `Button` `type`) |
| `theme` | `'solid' \| 'light' \| 'outline' \| …` | `'solid'` | Button theme (any `Button` `theme`) |
| `icon` | `ReactNode` | -- | Icon element |
| `iconPosition` | `'left' \| 'right'` | `'left'` | Icon placement |
| `onClick` | `(e) => void` | -- | Click handler |

Any other [`Button`](/docs/components/base/button) prop (`loading`, `block`, …) is also accepted.
