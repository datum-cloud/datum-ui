import { AvatarStack } from '@/components/ui'

## Overview

AvatarStack renders a row of overlapping avatar images with automatic overflow handling. When the number of avatars exceeds `maxAvatarsAmount`, a "+N" indicator appears showing the count of hidden avatars. Each avatar displays a tooltip with the user's name on hover.

## Dependencies

This component requires the following packages:

```package-install
@radix-ui/react-avatar
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `avatars` | `{ name: string, image: string }[]` | -- | Array of avatar objects with name and image URL |
| `maxAvatarsAmount` | `number` | `3` | Maximum number of avatars to display before showing overflow |
| `orientation` | `'vertical' \| 'horizontal'` | `'vertical'` | Layout direction of the avatar stack |
| `spacing` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Overlap amount between avatars |
| `avatarClassName` | `string` | -- | Additional CSS classes applied to each avatar |
| `className` | `string` | -- | Additional CSS classes for the container |
| `...props` | `React.HTMLAttributes<HTMLDivElement>` | -- | All standard HTML div attributes |

### avatarStackVariants

The `avatarStackVariants` utility is also exported for use cases where you need the stack layout styling without the full component.

```tsx
import { avatarStackVariants } from '@datum-cloud/datum-ui/avatar-stack'

<div className={avatarStackVariants({ orientation: 'vertical', spacing: 'md' })}>
  {/* custom avatar elements */}
</div>
```
