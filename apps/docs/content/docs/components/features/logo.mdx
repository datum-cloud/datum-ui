## Overview

`Logo` is a compound namespace exposing four SVG components:

- `Logo.Flat` тАФ horizontal lockup (D mark + Datum wordmark)
- `Logo.Stacked` тАФ vertical lockup (D mark above the wordmark)
- `Logo.Icon` тАФ the standalone D mark
- `Logo.Text` тАФ the "Datum" wordmark only

Each component accepts a `tone` prop with four values:

| Tone          | Use case                                                         |
| ------------- | ---------------------------------------------------------------- |
| `brand`       | Default. Navy text + rose mark тАФ for light surfaces.             |
| `mono-light`  | Lime mark + white text тАФ the standard dark-surface treatment.    |
| `mono-dark`   | Solid navy, monochrome тАФ for light surfaces, no rose accent.     |
| `white`       | Solid white тАФ for photographic or strongly coloured surfaces where the lime accent would clash. |

Sizing is driven entirely by Tailwind classes on `className` (`h-8 w-auto`, `size-12`, etc.). All three components default to `role="img"` with `aria-label="Datum"`; pass `decorative` to mark a logo as decorative (sets `aria-hidden`).

## Theme-aware adapter

For apps that already use Datum's theme system, `@datum-cloud/datum-ui/logo/themed` exports `ThemedLogo.{Flat,Stacked,Icon}` which pick the appropriate tone from `useTheme()` (and fall back to `brand` during SSR via `ClientOnly`):

```tsx
import { ThemedLogo } from '@datum-cloud/datum-ui/logo/themed'

<ThemedLogo.Flat className="h-8 w-auto" />
//                renders `brand` on light theme, `mono-light` on dark theme
```

Consumers without a theme system should use the core `Logo.*` components directly тАФ no theme peer dependency required.

## Accessibility

- Default `role="img"` with `aria-label="Datum"`.
- Override the label with `aria-label="тАж"` for branded contexts (e.g. "Datum Cloud").
- Pass `decorative` when the logo sits next to a textual brand name; this drops the role/label and sets `aria-hidden="true"`.

```tsx
<Logo.Flat aria-label="Datum Cloud" className="h-8 w-auto" />
<Logo.Icon decorative className="size-6" /> {/* sibling to "Datum" text */}
```
