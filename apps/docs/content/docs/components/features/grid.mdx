## Overview

The Grid system provides a 24-column layout using `Row` and `Col` components. Rows manage horizontal spacing and alignment, while columns define width spans and support responsive breakpoints (xs, sm, md, lg, xl, xxl) for adaptive layouts.

## Dependencies

No additional dependencies required — this component uses only React and Tailwind CSS.

## Setup

Import the grid CSS in your app's stylesheet:

```css title="app.css"
@import '@datum-cloud/datum-ui/grid';
```

## Row Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'flex'` | -- | Enables flexbox layout on the row |
| `align` | `'top' \| 'middle' \| 'bottom'` | -- | Vertical alignment of columns (requires `type="flex"`) |
| `justify` | `'start' \| 'end' \| 'center' \| 'space-around' \| 'space-between'` | -- | Horizontal distribution of columns (requires `type="flex"`) |
| `gutter` | `number \| Record<Breakpoint, number> \| [Gutter, Gutter]` | `0` | Spacing between columns in pixels. Pass a tuple for `[horizontal, vertical]` gutters |
| `className` | `string` | -- | Additional CSS classes |
| `style` | `CSSProperties` | -- | Inline styles |
| `prefixCls` | `string` | `'grid'` | CSS class prefix |

## Col Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `span` | `number` | -- | Number of columns to span (out of 24) |
| `offset` | `number` | -- | Number of columns to offset from the left |
| `order` | `number` | -- | Flex order of the column |
| `push` | `number` | -- | Number of columns to push right |
| `pull` | `number` | -- | Number of columns to pull left |
| `xs` | `number \| ColSize` | -- | Span or config for screens < 576px |
| `sm` | `number \| ColSize` | -- | Span or config for screens >= 576px |
| `md` | `number \| ColSize` | -- | Span or config for screens >= 768px |
| `lg` | `number \| ColSize` | -- | Span or config for screens >= 992px |
| `xl` | `number \| ColSize` | -- | Span or config for screens >= 1200px |
| `xxl` | `number \| ColSize` | -- | Span or config for screens >= 1600px |
| `className` | `string` | -- | Additional CSS classes |
| `style` | `CSSProperties` | -- | Inline styles |
| `prefixCls` | `string` | `'grid'` | CSS class prefix |

### ColSize Object

When passing an object to a responsive breakpoint prop:

| Prop | Type | Description |
|------|------|-------------|
| `span` | `number` | Columns to span |
| `order` | `number` | Flex order |
| `offset` | `number` | Columns to offset |
| `push` | `number` | Columns to push |
| `pull` | `number` | Columns to pull |
